New Shopify Embedded App Setup

This is the modern official path for a Shopify app that opens inside Shopify Admin. The recommended scaffold is a React Router app created with Shopify CLI.

Before You Start

  • Install Node.js 20.10+ on your machine.
  • Install Git.
  • Make sure you have a Shopify Partner account.
  • Make sure you have a development store to test the app.

Install Shopify CLI

  • Install Shopify CLI globally: 
    npm install -g @shopify/cli@latest
  • Confirm the installation: 
    shopify version

Create the App

  • Move to the folder where you want the new app: 
    cd "C:\shopify apps"
  • Create the app: 
    shopify app init
  • Enter your new app name when Shopify asks.
  • Select Build a React Router app when prompted.

Connect the App

  • Open the new app folder: 
    cd your-new-app
  • Start local development: 
    shopify app dev
  • Log in to your Shopify Partner account if prompted.
  • Select your Partner organization.
  • Select the development store where you want to test the app.
  • Let Shopify CLI create the tunnel and app URLs automatically.

Confirm It Is Embedded

  • Open shopify.app.toml.
  • Make sure embedded = true is present.
  • Keep only the scopes your app really needs.
  • Make sure the generated app URL and redirect URLs are correct.
Embedded means the app opens inside Shopify Admin, usually inside an iframe.

First Build Goal

  • Keep the app clean and new in a separate folder.
  • Do not copy old business logic from your existing app at the start.
  • Build your first page inside Shopify Admin using Polaris and App Bridge.
  • Add new scopes, APIs, extensions, and features only when you need them.
